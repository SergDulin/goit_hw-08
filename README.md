# Поздоровлення с днем народження

Цей скрипт призначений для виведення списку колег, яких потрібно привітати з днем народження на наступному тижні.

## Використання

Додайте список словників users, кожен словник у ньому обов'язково має ключі name та birthday. Така структура представляє модель списку користувачів з їх іменами та днями народження. name — це рядок з ім'ям користувача, а birthday — це datetime об'єкт, в якому записаний день народження.

## Опис роботи

Функція get_birthdays_per_week(users) приймає список користувачів (users) і виводить на екран імена користувачів, у яких є дні народження наступного тижня, розподілені за днями тижня. Користувачі, у яких день народження на вихідних, потраплять у список с привітанням в понеділок.

## Важливо

Приклад списку користувачів users, в форматі списка словників:

users = [
        {'name': 'Bill', 'birthday': datetime(1975, 8, 16)},
        {'name': 'Jill', 'birthday': datetime(2003, 7, 18)},
        {'name': 'Kim', 'birthday': datetime(1993, 6, 25)},
        {'name': 'Jan', 'birthday': datetime(1087, 6, 29)},
        {'name': 'Alex', 'birthday': datetime(2006, 6, 24)},
        {'name': 'Serhii', 'birthday': datetime(2001, 7, 1)},
    ]
